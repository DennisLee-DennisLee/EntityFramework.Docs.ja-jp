# <a name="ef-core-tools--extensions"></a>EF Core のツールと拡張機能

ツールと拡張機能は、Entity Framework Core の追加機能を提供します。

拡張機能はさまざまなソースによってビルドされます。 一部の拡張機能は、Entity Framework Core プロジェクトの一部として保守管理されていません。 サードパーティ拡張機能の利用を検討するとき、要件に合うよう、品質、使用許諾、サポートなどを必ず評価してください。
